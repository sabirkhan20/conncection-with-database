# Hospital Management System (Python & MySQL)

This project is a simple **Hospital Management System** built using **Python** and **MySQL**.  
It demonstrates how Python can interact with a MySQL database to store, retrieve, and manage hospital-related data.

The project focuses on performing basic database operations such as creating tables, inserting records, fetching data, and managing database connections using Python.
